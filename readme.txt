Scalable Fault Resolution in Autonomous Networks via Generative Augmentation and Soft-Clustered Retrieval

Abstract: Efficient fault resolution is a key requirement for zero-touch autonomous network management across FCAPS functions. Recent retrieval-augmented approaches leverage historical fault-resolution repositories to guide the resolution of newly observed faults, but face a fundamental scalability challenge: larger knowledge bases improve coverage of diverse fault scenarios but increase retrieval latency, while smaller repositories enable faster retrieval at the cost of reduced resolution quality; limiting practical deployment in real-time network operations. This paper presents SCALA (Scalable LLM-Augmented Fault Resolution), a framework designed to jointly improve knowledge coverage and retrieval efficiency. SCALA first enriches the fault repository through GAN-based generative augmentation, increasing fault-resolution diversity and mitigating data imbalance. The augmented knowledge base is then organised using probabilistic soft clustering, enabling incoming fault queries to be directed to a semantically relevant subset of the repository rather than the entire search space. The retrieved context is subsequently leveraged by an LLM to generate fault resolution recommendations. This hierarchical retrieval strategy significantly reduces retrieval overhead while preserving contextual relevance. Evaluated across diverse fault scenarios and varying knowledge base sizes, SCALA improves fault resolution success rate by 57.55% over conventional RAG and 14.81% over knowledge-driven baselines, while reducing inference latency by 38.82%, which translates into faster fault recovery and reduced service degradation duration, establishing SCALA as a practical solution for scalable autonomous network fault management.

The source code for the GAN, GMM, and RAG components used in this work is provided in this repository. Please install the required dependencies listed in requirements.txt before running the code.

The file fault-logs-o-ran-64.xls contains the original fault log corpus, which is used as input to the GAN for generating synthetic fault data. Once the knowledge base is expanded using the generated data, the GMM and RAG modules can be executed to perform clustering and enable intelligent fault resolution on new inference data.
